🎯 KULLANIM SENARYOLARI

SENARYO 1: Tek Hesap, Bir Grup
  1. API ayarlarını bir kez kaydet
  2. Grup ID'sini kaydet
  3. Dosya yükle → Başla
  4. Done!

SENARYO 2: Çoklu Grup Ekleme
  1. Grup 1'e ekle
  2. Ayarlar → Grup ID'sini değiştir
  3. Tekrar dosya yükle → Başla
  4. Sıradaki grup... tekrar et

SENARYO 3: Farklı Hesaplar
  1. Hesap 1 API ayarlarını gir
  2. Bağlan → Ekleme yap
  3. Disconnect
  4. API ayarlarını değiştir (Hesap 2)
  5. Tekrar bağlan → Yeni hesapla ekleme yap
