═══════════════════════════════════════════════════════════════════════════════
                            🎯 SISTEM ÖZETİ
═══════════════════════════════════════════════════════════════════════════════

NE YAPAR?
─────────
✅ Web paneli üzerinden Telegram grubunuza/kanalınıza toplu olarak üye ekler
✅ Kullanıcı adlarını dosyadan (TXT, CSV, XLSX) yükler
✅ Her kullanıcı arasında 30 saniye bekleme yaparak ekleme işlemi gerçekleştirir
✅ Real-time ilerleme çubuğu ile işlemi takip ettirir
✅ Detaylı sonuç raporlaması gösterir (başarılı, başarısız, engellenen)

NASIL ÇALIŞIR?
──────────────
1. Web paneline giriş → "Telegram'a Bağlan" tıklama
2. SMS kodu girişi
3. Kullanıcı listesi dosyası yükleme
4. "Eklemeye Başla" tıklama
5. Her 30 saniyede bir kullanıcı ekleme
6. İşlem tamamlanmıştı → Sonuç raporu

═══════════════════════════════════════════════════════════════════════════════
                        ⚡ 5 DAKİKALIK BAŞLANGIÇ
═══════════════════════════════════════════════════════════════════════════════

ADIM 1: API BİLGİLERİ AL (2 dakika)
───────────────────────────────────
1. https://my.telegram.org/apps adresine git
2. Telefon numaranla giriş yap
3. "Create Application" tıkla
4. Formu doldur ve API_ID + API_HASH'i kopyala

ADIM 2: CONFIG DOSYASINI DÜZENLE (1 dakika)
──────────────────────────────────────────
config.py dosyasını aç ve bunları gir:

   API_ID = 12345678
   API_HASH = "abcdef123456..."
   PHONE_NUMBER = "+905551234567"
   TARGET_GROUP_ID = -1001234567890

ℹ️ TARGET_GROUP_ID nasıl bulunur?
   Grubun linkini kopyala: https://t.me/c/1234567890/
   → Chat ID: -1001234567890 ✓

ADIM 3: BAŞLAT (1 dakika)
────────────────────────
Windows:
   run.bat

Linux/Mac:
   bash run.sh

Veya manuel:
   python -m venv venv
   .\venv\Scripts\activate.ps1  (Windows)
   pip install -r requirements.txt
   python main.py

ADIM 4: WEB PANELİ AÇ (1 saniye)
───────────────────────────────
   http://localhost:5000

ADIM 5: KULLANICILAR EKLE (5 saniye)
───────────────────────────────────
   1. "Telegram'a Bağlan" tıkla
   2. SMS kodu gir
   3. Dosya yükle (TXT/CSV/XLSX)
   4. "Eklemeye Başla" tıkla
   5. ✅ HAZIR!

BACKEND (Python + Flask):
─────────────────────────
• Flask 3.0.0 - Web framework
• Telethon 1.33.5 - Telegram API istemcisi
• AsyncIO - Hızlı asynchronous işlem
• Server-Sent Events (SSE) - Real-time stream
• Session yönetimi - Telegram oturumu
• Error handling - Flood control, Privacy, vb

FRONTEND (HTML5 + CSS3 + JavaScript):
──────────────────────────────────────
• Modern responsive design (Mobil uyumlu)
• Sürükle-bırak dosya yükleme
• Real-time ilerleme animasyonları
• SSE stream okuma
• Dinamik DOM güncelleme
• 100% vanilla JavaScript (Framework yok)

ÖZEL ÖZELLİKLER:
────────────────

✓ Otomatik retry (Flood kontrol)
✓ Kullanıcı gizlilik kontrolü
✓ Duplikat temizleme
✓ CSV/XLSX/TXT desteği
✓ Türkçe hata mesajları
✓ Unicode desteği
