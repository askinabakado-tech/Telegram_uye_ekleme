ADIM 1️⃣  - API BİLGİLERİ AL (2 dakika)
  • https://my.telegram.org/apps git
  • Giriş yap
  • "Create Application" tıkla
  • API_ID ve API_HASH'i kopyala

ADIM 2️⃣  - CONFIG DOSYASINI DÜZENLE (1 dakika)
  • config.py dosyasını aç
  • Şunları gir:
    ✓ API_ID
    ✓ API_HASH
    ✓ PHONE_NUMBER (+905551234567)
    ✓ TARGET_GROUP_ID

ADIM 3️⃣  - PROGRAMI BAŞLAT (1 dakika)
  
  Windows'ta:
  $ run.bat
  
  Linux/Mac'te:
  $ bash run.sh
  
  Veya manuel:
  $ python -m venv venv
  $ .\venv\Scripts\activate.ps1  (Windows)
  $ pip install -r requirements.txt
  $ python main.py

ADIM 4️⃣  - WEB PANELİNİ AÇ (1 saniye)
  • Tarayıcı otomatik açılacak
  • Veya: http://localhost:5000

ADIM 5️⃣  - KULLANICILAR EKLE (5 saniye)
  1. "Telegram'a Bağlan" tıkla
  2. SMS kodu gir
  3. Dosya yükle (TXT/CSV/XLSX)
  4. "Eklemeye Başla" tıkla
  5. ✅ HAZIR!
